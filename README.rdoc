= Gitable

Like Addressable::URI but for Git

require 'gitable/uri'
uri = Gitable::URI.parse('git@github.com:martinemde/gitable.git')

uri.path # => 'martinemde/gitable.git'
uri.user # => 'git'
uri.host # => 'github.com'

Works with any valid Git uri.
